# Тестовое задание для Mindbox

Исходный код для задания с вычислением площадей находится в папке [src](/src/).

Исходный код юнит-тестов находится в папке [tests](/tests/).

SQL-запрос для соответствующего задания находится в файле [SQLQuery.sql](SQLQuery.sql).

В рамках выполнения тестового задания использовались следующие дополнительные библиотеки:

1. [FluentAssertions](https://fluentassertions.com/ "Официальный сайт FluentAssertions") (библиотека для более удобного ассертинга при юнит-тестировании)
